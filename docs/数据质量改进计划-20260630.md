### 阶段2：工具改进（2周内）

**目标**：提升数据提取准确率

#### 改进方案1：优化OCR提取脚本

**问题**：当前脚本提取成功率低

**改进点**：

1. **增加金额提取规则**
```python
# 新增金额提取模式
amount_patterns = [
    r'合同总金额[：:]\s*人民币\s*(\d+(?:,\d{3})*(?:\.\d+)?)\s*元',
    r'总金额[：:]\s*(\d+(?:,\d{3})*(?:\.\d+)?)\s*元',
    r'合同价款[：:]\s*(\d+(?:,\d{3})*(?:\.\d+)?)\s*元',
    r'金额[：:]\s*(\d+(?:,\d{3})*(?:\.\d+)?)\s*元',
    r'(\d+(?:,\d{3})*)\s*元.*?(?:大写|合计|总计)',
]
```

2. **增加项目编号提取规则**
```python
# 新增编号提取模式
id_patterns = [
    r'(ZH02-\d{7,9})',
    r'(SGSC[A-Z]{2,4}\d{8,10})',
    r'合同编号[：:]\s*([A-Z0-9\-]+)',
    r'编号[：:]\s*([A-Z0-9\-]+)',
]
```

3. **增加甲方乙方提取规则**
```python
# 新增甲乙方提取模式
party_patterns = [
    r'甲方[：:]\s*(.+?)(?:\s*$|\s*乙方)',
    r'乙方[：:]\s*(.+?)(?:\s*$|\s*签订)',
]
```

**实施步骤**：
1. 更新提取脚本
2. 重新提取30个文件
3. 对比提取结果
4. 调整优化

**预期效果**：提取准确率从 0% → 50%
