### 2. 清理静态路由合并逻辑

**文件**: `ui-vben/apps/web-antd/src/router/routes/index.ts`

**修改前**:
```typescript
const accessRoutes = [...dynamicRoutes, ...staticRoutes];  // ❌ 合并所有静态路由
```

**修改后**:
```typescript
// 只保留旧路由重定向，业务路由由后端动态生成
const legacyRedirectRoutes = dynamicRoutes.filter(
  (route) => route.name?.toString().startsWith('Legacy'),
);
const accessRoutes: RouteRecordRaw[] = [...legacyRedirectRoutes];
```

**效果**: 后端菜单成为唯一路由真值，避免静态路由和动态路由冲突。
