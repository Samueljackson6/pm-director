### 供应商菜单路径 Bug（根因确认）

**位置**：`backend/routers/auth.py:226`

```python
{
    'name': '供应商列表',
    'path': '/suppliers',  # ❌ 绝对路径，应该是相对路径 'list'
    'component': '/suppliers/index',
}
```

**影响**：
- 父路径 `/suppliers` + 子路径 `/suppliers` = `/suppliers/suppliers` 或 `/suppliers//suppliers`
- 导致供应商菜单 404

**修复**：将 `path: '/suppliers'` 改为 `path: 'list'`
