## 🔍 RuoYi框架分析

### RuoYi框架概况

**RuoYi是一套基于Spring Boot的后台管理系统框架**，有多个版本：

| 版本 | 技术栈 | 适用场景 |
|------|--------|---------|
| **RuoYi** | Spring Boot + Thymeleaf | 传统单体应用 |
| **RuoYi-Vue** | Spring Boot + Vue.js | 前后端分离 |
| **RuoYi-Cloud** | Spring Cloud + Vue.js | 微服务架构 |
| **RuoYi-App** | Spring Boot + 移动端 | 移动端应用 |

**核心特点**：
- ✅ 开源免费（MIT协议）
- ✅ 功能齐全（权限管理、代码生成、系统监控）
- ✅ 社区活跃（Gitee星标40k+，GitHub星标25k+）
- ✅ 文档完善（官方文档、视频教程）
- ✅ 易于扩展（模块化设计）

### 数据迁移方案

**SQLite → MySQL迁移**：

```sql
-- 方案1：直接迁移（推荐）
-- 使用SQLite导出CSV → MySQL导入

-- 方案2：脚本迁移
-- Python脚本读取SQLite → 写入MySQL

-- 方案3：数据库工具
-- 使用DBeaver等工具直接迁移
```

**迁移脚本示例**：
```python
import sqlite3
import mysql.connector

# SQLite连接
sqlite_conn = sqlite3.connect('database/project_management.db')

# MySQL连接
mysql_conn = mysql.connector.connect(
    host='localhost',
    user='root',
    password='password',
    database='ruoyi_office'
)

# 迁移contracts表
sqlite_cursor = sqlite_conn.cursor()
mysql_cursor = mysql_conn.cursor()

sqlite_cursor.execute('SELECT * FROM contracts')
for row in sqlite_cursor.fetchall():
    mysql_cursor.execute(
        'INSERT INTO sys_contract VALUES (...)',
        row
    )

mysql_conn.commit()
```
