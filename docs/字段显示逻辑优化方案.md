# 字段显示逻辑优化方案

> **创建时间**：2026-06-24  
> **维护人**：pm-director

---

## 一、优化目标

**核心目标**：提升用户体验，减少冗余信息显示

**关键原则**：
- 相关字段就近显示
- 冗余字段条件隐藏
- 重要字段突出显示

---

## 二、字段分组方案

### 2.1 基本信息（顶部）

**字段列表**：
1. 项目编号
2. 项目名称
3. 合同类型（新增）
4. 客户名称
5. 客户类型（新增）
6. 项目状态
7. 项目金额

**显示逻辑**：始终显示

---

### 2.2 合同信息（中部）

**字段列表**：
1. 合同签订日期
2. 合同金额
3. 不含税金额
4. 税率
5. 税额
6. 合同信息（关联字段）

**显示逻辑**：始终显示

---

### 2.3 技术信息（中部）

**字段列表**：
1. 技术方向（新增，仅科技项目显示）
2. 技术负责人
3. 项目简介

**显示逻辑**：
```
IF 合同类型 = "科技项目" THEN
  显示"技术方向"字段
ELSE
  隐藏"技术方向"字段
END IF
```

---

### 2.4 客户信息（中部）

**字段列表**：
1. 客户属性
2. 客户部门（条件显示）
3. 区域
4. 甲方联系人
5. 联系电话

**显示逻辑**：
```
IF 客户类型 = "电网客户" THEN
  显示"客户部门"字段
ELSE
  隐藏"客户部门"字段
END IF
```

---

### 2.5 团队信息（中部）

**字段列表**：
1. 商务负责人
2. 项目经理
3. 主要人员

**显示逻辑**：始终显示

---

### 2.6 进度信息（下部）

**字段列表**：
1. 进度百分比
2. 预计交付日期
3. 实际交付日期
4. 阶段进度（关联字段）

**显示逻辑**：始终显示

---

### 2.7 财务信息（下部）

**字段列表**：
1. 已开票
2. 回款金额
3. 回款进度
4. 回款跟踪（关联字段）

**显示逻辑**：始终显示

---

### 2.8 沟通记录（下部）

**字段列表**：
1. 最后沟通记录
2. 历史沟通记录
3. 待办事项

**显示逻辑**：始终显示

---

### 2.9 风险管理（底部）

**字段列表**：
1. 风险等级
2. 风险说明

**显示逻辑**：始终显示

---

### 2.10 分包信息（底部，条件显示）

**字段列表**：
1. 分包单位
2. 分包单位联系人
3. 分包合同金额
4. 分包合同签订时间

**显示逻辑**：
```
IF 分包单位不为空 THEN
  显示分包信息字段组
ELSE
  隐藏分包信息字段组
END IF
```

## 三、字段颜色标记

### 3.1 状态颜色

| 项目状态 | 颜色 | 说明 |
|---------|------|------|
| 已交付已验收 | 🟢 绿色 | 已完成 |
| 已中标交付中 | 🔵 蓝色 | 进行中 |
| 已全额开票 | 🟡 黄色 | 待回款 |
| 已中标未签订 | 🟠 橙色 | 风险提示 |
| 商机跟踪 | ⚪ 灰色 | 早期阶段 |

---

### 3.2 风险颜色

| 风险等级 | 颜色 | 说明 |
|---------|------|------|
| 低风险 | 🟢 绿色 | 正常 |
| 高风险 | 🔴 红色 | 需关注 |

---

### 3.3 进度颜色

| 进度百分比 | 颜色 | 说明 |
|-----------|------|------|
| 0-30% | 🟡 黄色 | 初期 |
| 31-70% | 🔵 蓝色 | 中期 |
| 71-99% | 🟢 绿色 | 后期 |
| 100% | ✅ 完成标记 | 已完成 |

## 四、字段排序优化

### 4.1 当前排序问题

**问题**：
- 相关字段分散
- 重要字段不够突出
- 逻辑不够清晰

---

1. 基本信息 → 合同信息 → 项目信息 → 财务信息 → 沟通记录
2. 重要字段前置
3. 关联字段就近显示

```
10. 技术负责人
11. 商务负责人
12. 项目经理
13. 区域
14. 客户部门（条件显示）
15. 甲方联系人
16. 联系电话
17. 进度百分比
18. 预计交付日期
19. 实际交付日期
20. 已开票
21. 回款金额
22. 最后沟通记录

...
```

## 六、实施步骤

### 步骤1：字段顺序调整
- 在飞书多维表格中调整字段顺序
- 确保相关字段就近显示

### 步骤2：条件显示配置
- 检查飞书是否支持字段条件显示
- 如支持，配置显示规则
- 如不支持，更新字段说明

### 步骤3：视图创建
- 创建默认视图、详细视图、财务视图、进度视图
- 为每个视图配置显示字段

### 步骤4：颜色标记
- 配置项目状态、风险等级的颜色
- 提升视觉识别效率

### 步骤5：测试验证
- 创建测试记录验证效果
- 收集团队反馈
- 持续优化

---

**维护说明**：
- 定期检查字段显示逻辑有效性
- 根据团队反馈持续优化
- 新增字段时更新显示逻辑
