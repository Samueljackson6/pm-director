## 二、手动配置步骤

### 步骤1：打开多维表格

**链接**：https://scnfxvn6tpnq.feishu.cn/base/HSPXbBE0qaemg4swVAQcchXKnMd

---

### 步骤2：找到字段

1. 点击表格右上角"字段设置"
2. 找到"合同类型"、"技术方向"、"客户类型"字段

---

### 步骤3：添加选项

**合同类型**：
1. 点击"合同类型"字段
2. 点击"编辑选项"
3. 依次添加：科技项目、技术服务、工程服务、咨询服务、合作交付、自有能力
4. 点击"保存"

**技术方向**：
1. 点击"技术方向"字段
2. 点击"编辑选项"
3. 依次添加：人工智能、大数据、物联网、云计算、区块链、数字孪生、其他
4. 点击"保存"

**客户类型**：
1. 点击"客户类型"字段
2. 点击"编辑选项"
3. 依次添加：电网客户、企业客户、政府客户、其他
4. 点击"保存"
