## 二、页面设计

### 1. 综合仪表盘
- 总合同数 / 总金额 / 已开票 / 已回款 / 回款率 卡片
- 合同金额TOP10 (横向柱状图)
- 月度开票/回款趋势 (折线图)
- 项目类型分布 (饼图)

### 2. 合同台账
- 搜索/筛选/排序
- 项目编号/名称/甲方/金额/类型/进度

### 3. 合同详情
- 基本信息卡片
- 研究阶段进度条
- 付款计划列表
- 交付物清单

### 4. 财务追踪
- 开票vs回款对比图
- 供应商财务汇总

## 三、API设计

`
GET  /api/stats                  # 综合统计
GET  /api/contracts              # 合同列表
GET  /api/contracts/{id}         # 合同详情
GET  /api/contracts/{id}/stages  # 阶段
GET  /api/contracts/{id}/payments # 付款
GET  /api/contracts/{id}/deliverables # 交付物
GET  /api/finance/summary        # 财务汇总
GET  /api/finance/trend          # 财务趋势
`

## 四、实施计划

### P0 (1-2天)
- FastAPI后端 SQLite只读API
- Vue3仪表盘 MVP (总览卡片 + TOP10 + 趋势图)

### P1 (3-4天)
- 合同列表 + 详情页
- 财务追踪页
- 阶段进度可视化

### P2 (按需)
- RuoYi-Vben iframe集成
- SQLite→MySQL迁移
- RuoYi模块定制
