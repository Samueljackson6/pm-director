## 🎨 HTML生成阶段

### 章节1：核心数据卡片

**必须包含**：
- 进行中项目数
- 商机跟踪数
- 2026年已开票
- 2026年已回款 + **补充字段**（2025年开票+2026年开票）

**格式要求**：
- 4个卡片横向排列
- 数字大号加粗
- 补充字段小字灰色

### 章节2：各年度合同情况汇总

**必须包含**：
- 2024年、2025年、2026年、合计
- 每行：年份、项目数、合同金额、已开票、已回款、回款率

**格式要求**：
- 表格形式
- 2026年行高亮
- 合计行黄色背景

### 章节3：本周进展相关项目

**筛选条件**：
- 本周有沟通记录更新的项目
- ❌ **不包含大额项目**（避免重复）

**每个项目必须显示**：
- 项目名称
- 客户单位
- 合同金额
- 商务经理
- 项目经理
- 供应商
- 已付款
- 已开票/已回款
- 本周进展（日期 + 内容）

**格式要求**：
- 卡片式布局
- 金额右对齐绿色
- 最新进展蓝色背景左边框

### 章节4：大额项目跟进汇总

**判定标准**：
- 合同金额 ≥ 200万元
- **特殊项目：HPLC项目（强制加入，金额显示"待补充"）**

**每个项目必须显示**：
- 项目名称、客户单位、合同金额
- 商务经理、项目经理
- 开票进度（百分比 + 金额）
- 回款进度（百分比 + 金额）
- **开票历史**：日期 + 金额（所有记录）
- **回款历史**：日期 + 金额（所有记录）
- **供应商付款**：供应商名称 + 分包合同 + 日期 + 金额（所有记录）
- 最新进展

**格式要求**：
- 卡片式布局
- 历史表格三列：开票历史、回款历史、供应商付款
- 大额项目合计：项目数量、合同金额、已开票、已回款

### 章节5：其余辅助类工作

**内容**：
- 财务数据更新与同步
- 商务拓展
- 其他辅助工作

**格式要求**：
- 分类列表
- 每类工作下列出具体事项

### 章节6：下周工作计划

**每个计划必须显示**：
- 项目名称
- 计划内容
- 商务经理
- 项目经理（为空显示"待定"）

**格式要求**：
- 卡片式布局
- 商务经理、项目经理字段必须显示

### 章节7：所有正在进行的项目清单

**按商务负责人分组**：

## 🔄 修正迭代阶段

### 审计不通过的处理流程

```
审计失败 → 定位问题 → 修正HTML → 重新审计 → 直到全部通过
```

### 常见问题及修正方法

#### 问题1：章节缺失
- **原因**：生成HTML时遗漏章节
- **修正**：重新生成完整HTML（从模板复制）

#### 问题2：2026年已回款补充字段缺失
- **原因**：忘记添加补充字段
- **修正**：手动添加补充字段HTML代码

#### 问题3：HPLC项目未添加
- **原因**：忘记添加特殊项目
- **修正**：在大额项目合计之前插入HPLC项目HTML

#### 问题4：负责人汇总信息格式错误
- **原因**：只显示项目数，缺少已开票/已回款
- **修正**：重新生成章节7 HTML（使用统计脚本）

#### 问题5：项目清单不完整
- **原因**：只显示部分项目
- **修正**：重新从财务表格读取完整项目列表
