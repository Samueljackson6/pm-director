## 📋 详细搭建步骤

### 步骤1：安装Docker（10分钟）

**安装方式**（推荐方式1）：

**方式1：Docker Desktop（推荐）**
```bash
# 访问官网下载
https://www.docker.com/products/docker-desktop/

# 或使用命令行安装（Ubuntu）
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# 将当前用户加入docker组
sudo usermod -aG docker $USER

# 重启系统或执行
newgrp docker
```

**验证安装**：
```bash
docker --version
docker run hello-world
```

---

### 步骤2：安装JDK 17（5分钟）

**方式1：使用apt安装（推荐，简单）**
```bash
sudo apt update
sudo apt install -y openjdk-17-jdk

# 验证安装
java -version
# 输出：openjdk version "17.x.x"
```

**方式2：使用SDKMAN（易管理多版本）**
```bash
# 安装SDKMAN
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"

# 安装JDK 17
sdk install java 17.0.9-tem

# 验证安装
java -version
```

**方式3：手动安装**
```bash
# 下载JDK 17
wget https://download.java.net/java/GA/jdk17.0.2/dfd4a8d0985749f896bed50d7138ee7f/8/GPL/openjdk-17.0.2_linux-x64_bin.tar.gz

# 解压到/usr/local
sudo tar -xzf openjdk-17.0.2_linux-x64_bin.tar.gz -C /usr/local/

# 配置环境变量
echo 'export JAVA_HOME=/usr/local/jdk-17.0.2' >> ~/.bashrc
echo 'export PATH=$JAVA_HOME/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# 验证安装
java -version
```

## ⚠️ 注意事项

### 1. Docker权限问题

**如果遇到权限错误**：
```bash
# 将当前用户加入docker组
sudo usermod -aG docker $USER

# 刷新组权限
newgrp docker

# 或重启系统
sudo reboot
```

---

### 2. MySQL初始化

**官方SQL文件获取**：
- 商业版：从 `ruoyi-office-db/dump/latest/` 获取
- 开源版：执行 `sql/mysql/dump-ruoyi-office-*.sql`
- 或加微信 17156169080 获取最新SQL文件

**执行SQL**：
```bash
# 连接MySQL
docker exec -it ruoyi-mysql mysql -u root -p123456

# 执行SQL文件
mysql> source /path/to/dump-ruoyi-office-*.sql
```

---

### 3. 端口冲突

**如果端口被占用**：
```bash
# 检查端口占用
sudo lsof -i :3306
sudo lsof -i :6379

# 停止占用端口的服务
sudo systemctl stop mysql
sudo systemctl stop redis

# 或修改docker-compose.yml中的端口映射
ports:
  - "33061:3306"  # 改为其他端口
  - "63791:6379"
```

---

### 4. 内存不足

**虽然您有15GB内存，但如果遇到内存不足**：
```bash
# 检查内存使用
free -h

# 清理缓存
sudo sync && sudo sysctl -w vm.drop_caches=3

# 增加交换空间（可选）
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```
