# 启动检查清单

> 每次启动时必须检查的铁律

---

## ⚠️ 飞书表格增量维护铁律

**违反后果**：数据永久丢失，无法恢复

### 强制操作流程

```
1. 读取当前值（调用 list/get）
2. 追加到历史字段
3. 更新当前字段
```

### 适用字段

- 最后沟通记录 / 历史沟通记录
- 回款进度
- 待办事项
- 备注

### 时间格式

- 日期：`YYYY-MM-DD`
- 回款：`YYYY-MM-DD：金额元`

### 自检清单

```
□ 我读取了当前值吗？
□ 我追加了历史记录吗？
□ 我用了标准时间格式吗？
□ 我保留了所有历史信息吗？
```
