# 阶段 3 UI 交互 Demo（历史探索稿）

> ⚠️ 本独立静态 Demo 已被用户评审否决，仅保留为历史探索证据，不再作为设计或生产实现依据。当前 Demo 位于 `ui-vben/apps/web-antd/src/views/dashboard/overview/phase3-review/`，验收见 `docs/audits/阶段3-Vben原生Demo验收-20260715.md`。

本 Demo 用于评审 PM Director 阶段 3 的信息架构、任务流和视觉方向。它是独立静态页面，不连接生产 API，不修改生产路由，业务内容均为脱敏模拟数据。

## 打开方式

可直接双击 `index.html` 打开。为了获得更稳定的浏览器行为，推荐在仓库根目录执行：

```powershell
python -m http.server 4173 --bind 127.0.0.1 --directory prototypes/phase3-ui-demo
```

然后访问：

```text
http://127.0.0.1:4173/#dashboard?view=all
```

## 评审范围

Demo 共覆盖 8 个可交互视图：

1. 驾驶舱：全局、经营、项目、财务四视图。
2. 合同台账：搜索、筛选、分页状态和详情入口。
3. 合同详情：履约链、阶段、财务、风险和关联项目。
4. 项目执行：项目队列、健康状态和里程碑。
5. 项目详情：阶段轨道、交付物和原路返回。
6. 客户应收：发票台账、回款记录和匹配状态。
7. 供应商档案：供应商履约队列。
8. 供应商详情：档案、供应商发票、付款记录和履约链。

## 可评审交互

- 单侧栏四任务域导航。
- 驾驶舱四视图写入 `view` 参数，刷新后保持。
- 驾驶舱行动项带筛选进入客户应收。
- 合同关键词筛选写入 URL。
- 合同详情进入项目详情后逐级原路返回，并恢复合同筛选。
- 明暗主题切换。
- 顶部“评审状态”切换加载、空数据和错误恢复。
- `Ctrl + K` 快速搜索。
- 860px 以下侧栏转为抽屉，620px 以下履约链转为纵向时间线。
- 支持键盘焦点、跳到主要内容和 `prefers-reduced-motion`。

## 文件说明

| 文件 | 用途 |
|------|------|
| `index.html` | 应用壳层、导航和图标资源 |
| `styles.css` | 设计令牌、布局、主题、响应式和动效 |
| `app.js` | 模拟数据、页面渲染、路由状态和交互 |
| `test_demo.py` | Playwright 关键任务流与浏览器诊断测试 |

## 自动化验收

先启动静态服务器，再执行：

```powershell
$env:PHASE3_DEMO_URL = "http://127.0.0.1:4173"
python prototypes/phase3-ui-demo/test_demo.py
```

测试覆盖桌面端和 390px 移动端，并收集：

- 控制台错误；
- 页面错误；
- 失败请求；
- HTTP 4xx/5xx；
- 移动端页面横向溢出。

## 评审边界

- Demo 中的风险明细为模拟数据，因为当前后端只有待办计数，没有完整任务对象。
- Demo 中的规范路由使用静态 Hash 表达；生产实现仍需修改后端权限菜单和 Vben 路由。
- “新建”“导出”“标记已处理”只展示交互反馈，不写入业务数据。
- 用户评审通过前，不将本 Demo 直接接入生产构建。
